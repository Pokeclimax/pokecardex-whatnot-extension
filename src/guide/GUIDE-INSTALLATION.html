Pokecardex vers Whatnot

Installation locale, sans compte et sans serveur.

Chrome et Edge

  1. Extrayez extension-chrome-edge.zip.
  2. Ouvrez chrome://extensions ou edge://extensions.
  3. Activez le mode développeur.
  4. Cliquez sur Charger l’extension non empaquetée et choisissez le dossier contenant manifest.json.

Firefox

  1. Utilisez Firefox 140 ou une version plus récente.
  2. Ouvrez extension-firefox-signed.xpi avec Firefox.
  3. Acceptez l’installation.
  4. Ouvrez Pokecardex.

Utilisation

  1. Ouvrez une carte sur Pokecardex.
  2. Cliquez sur Ajouter à ma liste.
  3. Cliquez sur le logo de l’extension, puis sur Ouvrir ma liste.
  4. Vérifiez les réglages essentiels et exportez le CSV Whatnot.

Les cartes et les réglages restent uniquement dans votre navigateur.
